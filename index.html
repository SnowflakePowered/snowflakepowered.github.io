What is Snowflake?

Snowflake is a brand-new zero-config emulator frontend powered by HTML5.

No more spending hours getting even the most basic emulators to work; from the moment you start, you're ready to play with Snowflake's "batteries included" approach. You can import your games library from an existing frontend, or you can let Snowflake scrape your games and find out exactly which games you have.

Don't like how something works? Want to change the look of your frontend? Snowflake's powerful theming system and plugin architecture allows you to completely change how Snowflake looks and feels. Want to install a new emulator? Simply install it from Snowflake's plugin repository* and everything has already been configured; just get up and play!

Snowflake is still under development. Star our GitHub repository to keep up with more news from Snowflake.

*Plugin repository will be available upon official release
