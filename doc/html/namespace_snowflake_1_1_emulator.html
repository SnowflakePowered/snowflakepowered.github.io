interface  IEmulatorAssembly
 Represents the assembly that an IEmulatorBridge will bridge to. An Emulator Assembly does not need to be executable, it can be a callable library file such as a libretro core. As long as the game runs when the emulator bridge endpoints are called it is able to be used with Snowflake. More...
 
interface  IEmulatorBridge
 Represents an Emulator Bridge An Emulator Bridge 'bridges' the gap between an unwrapper assembly, such as an .exe or a libretro core and wraps it so that Snowflake can call into it using specified endpoints. It is responsible for managing the controller and emulator configuration for the emulator, converting from Snowflake's config format into whatever format the assembly accepts. More...

enum Snowflake.Emulator.EmulatorAssemblyType

Types of emulator assemblies

Enumerator
EMULATOR_EXECUTABLE 

The assembly is an executable, such as an exe on Windows. retroarch.exe

would be an executable emulator assembly.

EMULATOR_LIBRARY 

The assembly is a library, and is not directly callable. If it is a libretro core, the emulator bridge is responsible for providing the libretro frontend environment.

EMULATOR_MISC 

The assembly is neither an executable nor a library. It is undefined.
