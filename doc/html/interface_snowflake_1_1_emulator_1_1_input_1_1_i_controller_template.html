Detailed Description

A Controller Mapping maps the buttons of a controller definition to the specific locations specified by an input template. For example, given an input template of

input_player{N}_select = "{RETROARCH_KEYBOARD_INPUT_SELECT}" A mapping of BTN_SELECT: RETROARCH_KEYBOARD_INPUT_SELECT would make whatever keyboard input entered for a controller's Select button to be set as input_player{N}_select, where {N} is replaced with the player index
