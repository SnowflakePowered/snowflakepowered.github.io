Detailed Description

This database stores which type of controller is to be used on which port. Snowflake supports up to 8 ports. It is recommended that when compiling controller profiles, the port number should indicate the slot of the controller profile to be used. Each port should correspond to a physical controller port on the console. Thus, player 1 should use the profile in slot 1 of the controller in port 1 Snowflake.Controller.IControllerDatabase
